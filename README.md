
# News App

# Stack

The News App is built using the following technologies and tools:

- **Android Kotlin
- **State Management:** Redux
- **Styling:** CSS Modules
- **API:** [News API](https://newsapi.org/)
- **Build Tool:** Webpack
- **Version Control:** Git
